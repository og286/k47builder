Unit images for the Konflikt 47 Army Builder.

Drop image files here (jpg/png/webp), then tell Claude the unit name and
the filename, or set it yourself via the ✎ button in a unit dropdown.

In the data, a unit's "photo" is just a path relative to index.html, e.g.
  "images/slammer-samuels.jpg"
Because it's a local file, it always loads and never breaks.
