# Unit image filenames

Drop a JPG named exactly as below into this `images/` folder and it appears on that unit.
If a file isn't present, the unit shows its tactical symbol instead.

Convention: `images/<unit-id>.jpg`. Override any single unit in-app with the ✎ button in a unit dropdown.
